# Safe Roblox Admin Panel Template

Этот репозиторий содержит **безопасный шаблон админ-панели для вашей собственной Roblox-игры**.

## Что внутри
- `roblox_admin_panel_safe.lua` — пример серверной админ-системы:
  - проверка прав по `UserId`;
  - команды: `fly`, `unfly`, `kill`, `trap`, `untrap`, `spawnbrainrot`;
  - серверная валидация аргументов;
  - защита от спама (cooldown);
  - создание `RemoteEvent` только на сервере.

## Важно
Этот шаблон предназначен только для **вашего собственного place/server-скрипта**.  
Не используйте его для обхода правил, взлома или вмешательства в чужие игры.
